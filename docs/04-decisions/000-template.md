## How to Use This Template

1. Copy this file as `NNN-short-name.md` (e.g., `006-caching-strategy.md`)
2. Number sequentially — **never reuse a number**, even if the decision is superseded
3. Fill in all sections — "Options Considered" is especially valuable for preventing re-evaluation
4. Set status to `Proposed` until the team agrees, then change to `Accepted`
5. **Never modify an old ADR.** If a decision is reversed, create a new ADR and mark the old one as `Superseded by ADR-NNN`
